# Tech Stack: San Rafael 360

Este documento especifica las versiones y herramientas utilizadas para evitar alucinaciones con sintaxis obsoleta o incompatibilidades.

## 📁 Tecnologías Principales

- **Runtime:** Node.js (>=20.0.0 <=24.x.x)
- **Framework Frontend:** Next.js 16.2.1 (App Router)
- **Headless CMS:** Strapi 5.40.0 (TypeScript)
- **Language:** TypeScript 5.x

## 🎨 Diseño y Animación

- **Styles:** Tailwind CSS v3.4+
- **Animations:** Framer Motion v11+
- **Icons:** Lucide React (v0.446)
- **Fonts:** Inter (Sans), Outfit (Heading)

## 🗃️ Base de Datos y Media

- **Local DB:** SQLite (`.tmp/data.db`)
- **Remote DB:** PostgreSQL (Railway)
- **Media Storage:** Local (`/public/uploads`)

## 🛠️ Herramientas de Desarrollo

- **Package Manager:** npm (>= 6.0)
- **CI/CD:** Vercel (Frontend), Railway (Backend)
- **Configuration:** Project Protocols (Atomic Coding, Plan-First)
