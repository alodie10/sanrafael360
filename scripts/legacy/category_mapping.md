# Propuesta de Mapeo de Categorías - San Rafael 360

Actualmente, el sitio WordPress tiene un conjunto de categorías limitado. A continuación presento cómo propongo mapear los "Rubros" del Excel a las categorías de "Listeo".

| Categoría en Excel | Categoría en WordPress (Destino) | Acción Recomendada |
| :--- | :--- | :--- |
| **Hoteles** | Alojamiento | Usar categoría existente. |
| **Cabañas** | Alojamiento | Usar categoría existente (¿O crear subcategoría "Cabañas"?). |
| **Apart Hoteles** | Alojamiento | Usar categoría existente. |
| **Hostels** | Alojamiento | Usar categoría existente. |
| **Posadas** | Alojamiento | Usar categoría existente. |
| **Gastronomía** | Gastronomía | Usar categoría existente. |
| **Bodegas** | Bodegas | Usar categoría existente. |
| **Agencias de Viaje** | Servicios | Usar categoría existente (¿O crear "Agencias de Viaje"?). |

### Sugerencias de mejora:
1.  **Subcategorías**: Podríamos crear subcategorías bajo "Alojamiento" para distinguir "Cabañas" de "Hoteles", esto ayuda al usuario en el filtro de búsqueda.
2.  **Agencias de Viaje**: A mi parecer, "Servicios" es muy genérico para una agencia de viajes. Recomiendo crear la categoría específica.

**Por favor, indícame si estás de acuerdo con este mapeo o si prefieres que cree categorías nuevas para alguno de estos rubros.**
